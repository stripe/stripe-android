val collectionOptions: AccountOnboardingProps.CollectionOptions?

Customizes collecting currently_due or eventually_due requirements and controls whether to include future requirements. Specifying eventually_due collects both eventually_due and currently_due requirements. Additionally, requirements can be used to collect only some requirements or exclude them.
