BecsDebitWidget

class BecsDebitWidget : FrameLayout

A form for accepting a customer's BECS account information.

A company name is required to render this widget. A company name can either be specified by passing it to the BecsDebitWidget constructor, or via XML:

<com.stripe.android.view.BecsDebitWidget
  android:id="@+id/element"
  android:layout_width="match_parent"
  android:layout_height="wrap_content"
  app:companyName="@string/becs_company_name" />
