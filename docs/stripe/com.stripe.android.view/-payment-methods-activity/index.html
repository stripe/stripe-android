class PaymentMethodsActivity : AppCompatActivity

An activity that allows a customer to select from their attached payment methods, or add a new one via AddPaymentMethodActivity.

This Activity is typically started through com.stripe.android.PaymentSession. To directly start this activity, use PaymentMethodsActivityStarter.startForResult.
