getPriceString

@JvmStatic fun getPriceString(price: Int, currency: Currency): String

Converts an integer price in the lowest currency denomination to a Google string value. For instance: (100L, USD) -> "1.00", but (100L, JPY) -> "100".

Parameters

price - the price in the lowest available currency denomination

currency - the Currency used to determine how many digits after the decimal

Return
a String that can be used as a Pay with Google price string
