Blocking method to create a PaymentMethod object. Do not call this on the UI thread or your app will crash.

See Create a PaymentMethod. POST /v1/payment_methods

Parameters

paymentMethodCreateParams - params with which to create the PaymentMethod

idempotencyKey - optional, see Idempotent Requests

stripeAccountId - Optional, the Connect account to associate with this request. By default, will use the Connect account that was used to instantiate the Stripe object, if specified.

Return
a PaymentMethod or null if a problem occurred
