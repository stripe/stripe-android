retrieveSourceSynchronous

@WorkerThread fun retrieveSourceSynchronous(sourceId: String, clientSecret: String): Source?

Retrieve an existing Source from the Stripe API. Do not call this on the UI thread or your app will crash.

See Retrieve a source. GET /v1/sources/:id

Parameters

sourceId - the Source.id field of the desired Source object

clientSecret - the Source.clientSecret field of the desired Source object

Exceptions

AuthenticationException - failure to properly authenticate yourself (check your key)

InvalidRequestException - your request has invalid parameters

APIConnectionException - failure to connect to Stripe's API

APIException - any other type of problem (for instance, a temporary issue with Stripe's servers)

Return
a Source if one could be found based on the input params, or null if no such Source could be found.
