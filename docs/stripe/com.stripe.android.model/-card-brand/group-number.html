groupNumber

fun groupNumber(cardNumber: String): Array<String?>
Content copied to clipboard

Separates a card number according to the brand requirements, including prefixes of card numbers, so that the groups can be easily displayed if the user is typing them in. Note that this does not verify that the card number is valid, or even that it is a number.

e.g. "4242424242424242" will return ["4242", "4242", "4242", "4242"]

Return

an array of strings with the number groups, in order. If the number is not complete, some of the array entries may be null.

Parameters

cardNumber

the raw card number
