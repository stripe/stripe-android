stripe / com.stripe.android.model / CardBrand / defaultMaxLength

defaultMaxLength

val defaultMaxLength: Int

The default max length when the card number is formatted without spaces (e.g. "4242424242424242")

Note that CardBrand.DinersClub's max length depends on the BIN (e.g. card number prefix). In the case of a CardBrand.DinersClub card, use getMaxLengthForCardNumber.
