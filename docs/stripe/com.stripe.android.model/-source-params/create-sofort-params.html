createSofortParams

@JvmStatic fun createSofortParams(amount: Long, returnUrl: String, country: String, statementDescriptor: String? = null): SourceParams

Create SOFORT Source params.

Parameters

amount - A positive integer in the smallest currency unit representing the amount to charge the customer (e.g., 1099 for a €10.99 payment).

returnUrl - The URL the customer should be redirected to after the authorization process.

country - The ISO-3166 2-letter country code of the customer’s bank.

statementDescriptor - A custom statement descriptor for the payment (optional).

Return
a SourceParams object that can be used to create a SOFORT source
