fun externalPaymentMethods(externalPaymentMethods: List<String>): PaymentSheet.Configuration.Builder

External payment methods to display in PaymentSheet.

If you specify any external payment methods here, you must also pass an ExternalPaymentMethodConfirmHandler to the FlowController or PaymentSheet constructor.

To learn more about external payment methods, see https://docs.stripe.com/payments/external-payment-methods?platform=android. For the full list of supported payment methods, see https://docs.stripe.com/payments/external-payment-methods?platform=android#available-external-payment-methods.
