onCreateIntent

abstract suspend fun onCreateIntent(paymentMethodId: String, shouldSavePaymentMethod: Boolean): CreateIntentResult

Called when the customer confirms the payment or setup.

Your implementation should create and confirm a PaymentIntent or SetupIntent on your server and return its client secret or an error if one occurred.

Parameters

paymentMethodId

The ID of the PaymentMethod representing the customer's payment details

shouldSavePaymentMethod

This is true if the customer selected the "Save this payment method for future use" checkbox. Set setup_future_usage on the PaymentIntent to off_session if this is true.
