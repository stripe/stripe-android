confirmPMFromCustomer

val confirmPMFromCustomer: Boolean?
Content copied to clipboard

This indicates if the payment method can be confirmed when attached to a customer and only the Payment Method id is available.

  • Null means that it is not supported, or that it is attached as a different type

  • false means that it is supported by the payment method, but not currently enabled (likely because of a lack of mandate support)

  • true means that a PM of this type attached to a customer can be confirmed
